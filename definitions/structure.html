Hexagonal Structure¶

set parent¶

Parent lattice can be defined by specifying twin mode. By this, the following properties are determined.

  1. twin indices

  2. shear strain functoin \(\gamma(r)\)

  3. parent matrix

FUTURE EDITED¶

First, supercell matrix \(\boldsymbol{M}\) is constructed as bellow

\[[\boldsymbol{m}^{p}, \boldsymbol{\eta}^{p}_1, \boldsymbol{\eta}^{p}_2] = [\boldsymbol{a}_1, \boldsymbol{a}_2, \boldsymbol{c}] \boldsymbol{M}.\]

where \(\boldsymbol{m}^{p}\), \(\boldsymbol{\eta}^{p}_1\) and \(\boldsymbol{\eta}^{p}_2\) are a little bit different from the ones in ‘get_twin_indices’ in twinmode.py because they are rescaled to be the vectors whose element are all integer. Therefore, every element of supercell matrix \(\boldsymbol{M}\) is integer. In this step, the positions of the lattice points in the original hexagonal lattice \(\boldsymbol{X}^{p}_{lat}\) and the relative positions of the atoms from each lattice point \(\boldsymbol{X}^{p}_{atm}\) are defined in \([\boldsymbol{m}^{p}, \boldsymbol{\eta}^{p}_1, \boldsymbol{\eta}^{p}_2]\) basis where

\[[\boldsymbol{a}_1, \boldsymbol{a}_2, \boldsymbol{c}] \boldsymbol{X}_{atm} = [\boldsymbol{m}^{p}, \boldsymbol{\eta}^{p}_1, \boldsymbol{\eta}^{p}_2] \boldsymbol{X}^{p}_{atm} = [\boldsymbol{a}_1, \boldsymbol{a}_2, \boldsymbol{c}] \boldsymbol{M} \boldsymbol{X}^{p}_{atm}\]

so

\[\boldsymbol{X}^{p}_{atm} = \boldsymbol{M}^{-1} \boldsymbol{X}_{atm}\]

In the next step, the cartesian basis vectors from \([\boldsymbol{e}_x, \boldsymbol{e}_y, \boldsymbol{e}_z]\) to \([\boldsymbol{e}_{\boldsymbol{m}^{p}}, \boldsymbol{e}_{\boldsymbol{\eta}^{p}_1}, \boldsymbol{e}_{\boldsymbol{k}^{p}_1}]\) where \(\boldsymbol{e}_{\boldsymbol{m}^{p}} = \frac{\boldsymbol{m}^{p}}{|\boldsymbol{m}^{p}|}\) \(\boldsymbol{e}_{\boldsymbol{\eta}^{p}_1} = \frac{\boldsymbol{\eta}^{p}_1}{|\boldsymbol{\eta}^{p}_1|}\) \(\boldsymbol{e}_{\boldsymbol{k}^{p}_1} = \frac{\boldsymbol{k}_1}{|\boldsymbol{k}_1|}\). In this step, rotation matrix \(\boldsymbol{R}\) is defined

\[[\boldsymbol{e}_{\boldsymbol{m}^{p}}, \boldsymbol{e}_{\boldsymbol{\eta}^{p}_1}, \boldsymbol{e}_{\boldsymbol{k}^{p}_1}] = [\boldsymbol{e}_x, \boldsymbol{e}_y, \boldsymbol{e}_z] \boldsymbol{R}\]

\(\boldsymbol{R}\) is orthogonal matrix.

\[\boldsymbol{R}^{-1} = \boldsymbol{R}^{T}\]

If the number vectors \(\boldsymbol{X}_{0}\) are transformed to \(\boldsymbol{X}^{p}_{0}\) by this operation, the following relations hold.

\[[\boldsymbol{e}_x, \boldsymbol{e}_y, \boldsymbol{e}_z] \boldsymbol{X}_{0} = [\boldsymbol{e}_{\boldsymbol{m}^{p}}, \boldsymbol{e}_{\boldsymbol{\eta}^{p}_1}, \boldsymbol{e}_{\boldsymbol{k}^{p}_1}] \boldsymbol{X}^{p}_{0} = [\boldsymbol{e}_x, \boldsymbol{e}_y, \boldsymbol{e}_z] \boldsymbol{R} \boldsymbol{X}^{p}_{0}\]

so

\[\boldsymbol{X}^{p}_{0} = \boldsymbol{R}^{-1} \boldsymbol{X}_{0} = \boldsymbol{R}^{T} \boldsymbol{X}_{0}\]

Be careful if you use def pymatgen.core.structure.Structure.apply_operation, its input rotation matrix is \(\boldsymbol{R}^{T}\), NOT \(\boldsymbol{R}\).

In the last step, parent lattice \(\boldsymbol{L}_p\) is defined as

\[[\boldsymbol{m}^{p}, \boldsymbol{\eta}^{p}_1, \boldsymbol{\eta}^{p}_2] = [\boldsymbol{a}_1, \boldsymbol{a}_2, \boldsymbol{c}] \boldsymbol{M} = [\boldsymbol{e}_x, \boldsymbol{e}_y, \boldsymbol{e}_z] \boldsymbol{H} \boldsymbol{M} = [\boldsymbol{e}_{\boldsymbol{m}^{p}}, \boldsymbol{e}_{\boldsymbol{\eta}^{p}_1}, \boldsymbol{e}_{\boldsymbol{k}^{p}_1}] \boldsymbol{R}^{-1} \boldsymbol{H} \boldsymbol{M} \equiv [\boldsymbol{e}_{\boldsymbol{m}^{p}}, \boldsymbol{e}_{\boldsymbol{\eta}^{p}_1}, \boldsymbol{e}_{\boldsymbol{k}^{p}_1}] \boldsymbol{L}_p\]

where

\[\boldsymbol{L}^p = \boldsymbol{R}^{-1} \boldsymbol{H} \boldsymbol{M}.\]

The coordinates of each points in \([\boldsymbol{m}^{p}, \boldsymbol{\eta}^{p}_1, \boldsymbol{\eta}^{p}_2]\) basis such as \(\boldsymbol{X}^{p}_{lat}\) and \(\boldsymbol{X}^{p}_{atm}\) DOES NOT be affected by this transformation.

HOWEVER, \(\boldsymbol{X}^{p}_{atm}\) may have to revise because two rigid atoms must be the nearest atoms from the specified \(\boldsymbol{K}_1\) plane. In the case you choose ‘c’ wyckoff letter, this revising (probably) always occur.

get_sheared_structure¶

When the shear ratio is \(\boldsymbol{r}\), the basis vectors of sheared structure are given as

\[[\boldsymbol{m}^{s}, \boldsymbol{\eta}^{s}_1, \boldsymbol{\eta}^{s}_2] = [\boldsymbol{m}^{p}, \boldsymbol{\eta}^{p}_1, \boldsymbol{\eta}^{p}_2 + \boldsymbol{d}]\]

where \(\boldsymbol{d}\) is given as

\[\boldsymbol{d} = r |\gamma(\boldsymbol{r})| (\boldsymbol{\eta}^{p}_2 \cdot \boldsymbol{e}_{\boldsymbol{k}^{p}_1}) \boldsymbol{e}_{\boldsymbol{\eta}^{p}_1} = r s \boldsymbol{e}_{\boldsymbol{\eta}^{p}_1}\]

where

\[s = |\gamma(\boldsymbol{r})| (\boldsymbol{\eta}^{p}_2 \cdot \boldsymbol{e}_{\boldsymbol{k}^{p}_1}).\]

Therefore, sheared lattice \(\boldsymbol{L}_s\) is given as

\[[\boldsymbol{m}^{s}, \boldsymbol{\eta}^{s}_1, \boldsymbol{\eta}^{s}_2] = [\boldsymbol{m}^{p}, \boldsymbol{\eta}^{p}_1, \boldsymbol{\eta}^{p}_2 + r s \boldsymbol{e}_{\boldsymbol{\eta}^{p}_1}] = [\boldsymbol{e}_{\boldsymbol{m}^{p}}, \boldsymbol{e}_{\boldsymbol{\eta}^{p}_1}, \boldsymbol{e}_{\boldsymbol{k}^{p}_1}] \boldsymbol{L}^s\]

where

\[\boldsymbol{L}^s = \boldsymbol{L}^p + r \boldsymbol{S}\]

where

\[\begin{split}\boldsymbol{S} = \begin{pmatrix} 0 & 0 & 0 \\ 0 & 0 & s \\ 0 & 0 & 0 \\ \end{pmatrix}.\end{split}\]

get_twin_structure¶

The operation from parent lattice to twin lattice \(\boldsymbol{W}^{t}\) is defineda as

\[[\boldsymbol{e}_{\boldsymbol{m}^{p}}, \boldsymbol{e}_{\boldsymbol{\eta}^{p}_1}, \boldsymbol{e}_{\boldsymbol{k}^{p}_1}] \boldsymbol{X}^{p}_{0} \longrightarrow [\boldsymbol{e}_{\boldsymbol{m}^{p}}, \boldsymbol{e}_{\boldsymbol{\eta}^{p}_1}, \boldsymbol{e}_{\boldsymbol{k}^{p}_1}] \boldsymbol{X}^{t}_{0}.\]

where

\[\boldsymbol{X}^{t}_{0} = \boldsymbol{W}^{t} \boldsymbol{X}^{p}_{0}.\]

In twin type \(\rm{I}\), rotation matrix \(\boldsymbol{W}^{t}\) is given as

\[\begin{split}\boldsymbol{W}^{t} = \begin{pmatrix} 1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & -1 \\ \end{pmatrix}.\end{split}\]

In twin type \(\rm{I\hspace{-1pt}I}\), rotation matrix \(\boldsymbol{W}^t\) is given as

\[\begin{split}\boldsymbol{W}^{t} = \begin{pmatrix} -1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & -1 \\ \end{pmatrix}.\end{split}\]

In both type, the following equation hold.

\[\boldsymbol{W}^{t,-1} = \boldsymbol{W}^{t,T} = \boldsymbol{W}\]

The relation between parent lattice \(\boldsymbol{L}^{p}\) and twin lattice \(\boldsymbol{L}^{t}\) is

\[[\boldsymbol{m}^{p}, \boldsymbol{\eta}^{p}_1, \boldsymbol{\eta}^{p}_2] = [\boldsymbol{e}_{\boldsymbol{m}^{p}}, \boldsymbol{e}_{\boldsymbol{\eta}^{p}_1}, \boldsymbol{e}_{\boldsymbol{k}^{p}_1}] \boldsymbol{L}^{p} \longrightarrow [\boldsymbol{e}_{\boldsymbol{m}^{p}}, \boldsymbol{e}_{\boldsymbol{\eta}^{p}_1}, \boldsymbol{e}_{\boldsymbol{k}^{p}_1}] \boldsymbol{W}^{t} \boldsymbol{L}^{p} \equiv [\boldsymbol{m}^{t}, \boldsymbol{\eta}^{t}_1, \boldsymbol{\eta}^{t}_2]\]

so

\[[\boldsymbol{m}^{t}, \boldsymbol{\eta}^{t}_1, \boldsymbol{\eta}^{t}_2] = [\boldsymbol{e}_{\boldsymbol{m}^{p}}, \boldsymbol{e}_{\boldsymbol{\eta}^{p}_1}, \boldsymbol{e}_{\boldsymbol{k}^{p}_1}] \boldsymbol{L}^{t}\]

where

\[\boldsymbol{L}^{t} = \boldsymbol{W}^{t} \boldsymbol{L}^{p}.\]

By this transformation, number vectors in the basis of parent vectors \(\boldsymbol{X}^{p}\) (including \(\boldsymbol{X}^{p}_{lat}\) and \(\boldsymbol{X}^{p}_{atm}\)) are not affected.

\[\begin{split}[\boldsymbol{m}^{p}, \boldsymbol{\eta}^{p}_1, \boldsymbol{\eta}^{p}_2] \boldsymbol{X}^{p} &= [\boldsymbol{e}_{\boldsymbol{m}^{p}}, \boldsymbol{e}_{\boldsymbol{\eta}^{p}_1}, \boldsymbol{e}_{\boldsymbol{k}^{p}_1}] \boldsymbol{L}^{p} \boldsymbol{X}^{p} \\ &\longrightarrow [\boldsymbol{e}_{\boldsymbol{m}^{p}}, \boldsymbol{e}_{\boldsymbol{\eta}^{p}_1}, \boldsymbol{e}_{\boldsymbol{k}^{p}_1}] \boldsymbol{W}^{t} \boldsymbol{L}^{p} \boldsymbol{X}^{p} \\ &= [\boldsymbol{m}^{t}, \boldsymbol{\eta}^{t}_1, \boldsymbol{\eta}^{t}_2] \boldsymbol{L}^{p,-1} \boldsymbol{W}^{t,-1} \boldsymbol{W}^{t} \boldsymbol{L}^{p} \boldsymbol{X}^{p} \\ &= [\boldsymbol{m}^{t}, \boldsymbol{\eta}^{t}_1, \boldsymbol{\eta}^{t}_2] \boldsymbol{X}^{p} \equiv [\boldsymbol{m}^{t}, \boldsymbol{\eta}^{t}_1, \boldsymbol{\eta}^{t}_2] \boldsymbol{X}^{t}\end{split}\]

so number vectors in the basis of twin vectors \(\boldsymbol{X}^{t}\) (including \(\boldsymbol{X}^{t}_{lat}\) and \(\boldsymbol{X}^{t}_{atm}\))

\[\boldsymbol{X}^{t} = \boldsymbol{X}^{p}\]

HexagonalTwinBoundary¶

__init__¶

To create ‘HexagonalTwinBoundary’ object, you have to specify the norm of a and c axis and its specie as a hexagonal metal information. Moreover, twinmode, twintype, dimension and x- y- shift respectively. If you set dimension equal ‘[1,1,2]’ and x-shift equal ‘1/2’, then parent and twin structures with its supecell as [1,1,2] and fix all the parent lattice point to -1/4 from its original points and fix all the twin lattice point to 1/4 from its original points. Then, dichromatic lattice \(\boldsymbol{L}^{d}\) is created.

\[[\boldsymbol{m}^{d}, \boldsymbol{\eta}^{d}_1, \boldsymbol{k}^{d}_1] = [\boldsymbol{e}_{\boldsymbol{m}^{p}}, \boldsymbol{e}_{\boldsymbol{\eta}^{p}_1}, \boldsymbol{e}_{\boldsymbol{k}^{p}_1}] \boldsymbol{L}^{d}\]

In the case ‘dim=[1,1,1]’, dichromatic lattice \(\boldsymbol{L}^{d}\) becomes

\[\begin{split}\boldsymbol{L}^{d} = \boldsymbol{L}^{s}(r=0.5) \begin{pmatrix} 1 & 0 & 0 \\ 0 & 1 & 0 \\ 0 & 0 & 2 \\ \end{pmatrix}.\end{split}\]

After this, number vectors \(\boldsymbol{X}^{p,t}\) in the bases both parent and twin are transformed into the dichromatic lattice frame.

\[\boldsymbol{L}^{p,t} \boldsymbol{X}^{p,t} = \boldsymbol{L}^{d} \boldsymbol{X}^{d}\]

so

\[\boldsymbol{X}^{d} = \boldsymbol{L}^{d -1} \boldsymbol{L}^{p,t} \boldsymbol{X}^{p,t}\]

get_sheared_structure¶

The twin boundary structure can be sheared by this function. Input ‘gamma’ represents shear strain (\(\gamma'\)). dichromatic lattice are sheared as

\[\boldsymbol{L}^{d, s} = \boldsymbol{L}^{d} + \boldsymbol{S}\]

where

\[\begin{split}\boldsymbol{S} = \begin{pmatrix} 0 & 0 & 0 \\ 0 & 0 & s' \\ 0 & 0 & 0 \\ \end{pmatrix}.\end{split}\]

where

\[s' = \gamma' |\boldsymbol{k}^{d}_1|\]
