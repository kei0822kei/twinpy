twinpy.common.utils.float2frac(var, accuracy=3, denominator=10)[source]¶

transform float to fractional (str object)

Parameters

  • var (float) – input value

  • accuracy (int) – threshold of whether two values are the same of not, In the case of ‘accuracy=3’, check the third decimal place.

  • denominator (int) – if ‘denominator=10’, check from 1/2 to 12/13

Returns

description

Return type

dict

Raises

ValueError – could not find fractional representation of input ‘var’

Examples

>>> float2frac(-1.333333, 3)
  '-4/3'
