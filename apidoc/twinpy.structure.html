This module deals with structures difference.

twinpy.structure.diff.get_structure_diff(cells: list, base_index: int = 0, include_base: bool = True) → dict[source]¶

Get structure diff with first cell in cells.

Parameters

  • cells – List of cells.

  • base_index – Base cell index.

  • include_base – If True, include base cell in output dict.

Returns

Containing ‘lattice_diffs’, ‘frac_posi_diffs’,

’cart_posi_diffs’ and ‘cart_norm_diffs’.

Return type

dict

Note

Return diff compared with base cell. ex. lattice_diffs[i] = ith_lattice - base_lattice. The value ‘frac_posi_diffs’ are the difference between two fractional coordinates which is not consider lattice change, which can be defined as ‘shuffle’. The value ‘cart_posi_diffs’ are the difference between two cartesian coordinates which is automatically consider lattice periodicity.
