twinpy.structure.diff module¶

compare structures

twinpy.structure.diff.get_structure_diff(cells: list, base_index: int = 0, include_base: bool = True) → dict[source]¶

Get structure diff with first cell in cells.

Parameters

  • cells (list) – list of cells

  • base_index (int) – base cell index

  • include_base (bool) – if True, include base cell in output dict

Returns

containing ‘lattice_diffs’, ‘scaled_posi_diffs’

and ‘cart_posi_diffs’

Return type

dict

Note

Return diff compared with base cell. ex. lattice_diffs[i] = ith_lattice - base_lattice. The value ‘scaled_posi_diffs’ are the difference between two fractional coordinates which is not consider lattice change. The value ‘cart_posi_diffs’ are the difference between two cartesian coordinates which is automatically consider lattice change.
